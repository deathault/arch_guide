Установка

Устанавливаем дистрибутив при помощи скрипта archinstall

После загрузки будет куча букв, где ОС будет загружаться в оперативку, но в конечном итоге покажет тебе такие сиськи:

Это значит что можно стартовать установку. Но тут есть нюанс - тебе нужна сеть. Если твое корыто оборудовано Ethernet портом то все ништяк. А если нет? А если у тебя ваще макбук с ебучим реалтеком? Мой те совет: берешь смартфон, врубаешь режим модема и подрубаешь в USB. Система сама его найдет и будет все загружать медленно но исправно.

и погнали.

Скрипт установки проверит новую версию, и запустить GUI установщик в терминале. Далее ничего сложного, тыкай да ставь нужные параметры, я же буду делать так как в табличке ниже:

ПараметрЧе ставить
Localesru_RU/UTF-8
Mirror regionRussia
Disk configurationbest-effort (default) \ выбираем свой диск \ ext4
BootloaderGrub
User accountAdd User - создаем нового пользователя и пароль \ соглашаемся сделать его superuser'ом \ Confirm and exit
Profile - TypeDesktop - Gnome
Profile - Graphics driverТут выбираем по своей железке, у меня RX 6700 XT, поэтому mesa
Profile - Greetergdm
Network configurationUse NetworkManager
Optional repositoriesmultilib

Далее жмем Install, одеваемся и идем за пивом. В конце установки он предложит установить доп пакеты, эта хуйня нужна, что бы сразу проставить драйвера на сеть (в основном), дабы все заработало после перезагрузки. Нам оно нахуй не надо, отвечаем No.
