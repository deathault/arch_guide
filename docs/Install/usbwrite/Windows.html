Windows

Запись Arch Linux на USB накопитель из-под Windows

Так, у нас имеется скачаный iso образ с дистрибутивом. Давайте его закатаем на флеху, внешний жесткий диск или че там у вас есть под рукой.

Для винды будем юзать Rufus. Пиздуем по ссылке, скачиваем любую версию и запускаем. Сразу жмем "Выбрать" и "Старт". Далее он спросит в каком режиме записать, оставьте как есть - в режиме iso. Может быть еще одно окно про Syslinux, ответьте "Да", ну и согласитесь с записью.

Сделали? Можем загружаться с флешки.
