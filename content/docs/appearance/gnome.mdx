## Gnome Tweaks

### Шрифты

Сначала установи шрифты. Я юзаю Roboto и Jetbrains mono для Моноширенного:

```sh title="Терминал"
yay -S ttf-roboto ttf-jetbrains-mono
```
| Параметр | Значение | 
| ----- | ----------- | 
| Текст интерфейса | Roboto Regular | 
| Текст документа | Roboto Regular | 
| Моноширенный текст | JetBrains Mono Regular |

### Внешний вид
Иконки Papirus из Aur:

```sh title="Иконки"
yay -S papirus-icon-theme
```
Сколько не пробовал тем, они либо отвратные, либо с косяками, либо отвалятся у тебя после обновления гнома. Короче юзай дефолт.
