## Gnome Tweaks

### Шрифты

Сначала установи шрифты. Я юзаю Roboto и Jetbrains mono для Моноширенного:

```sh title="Терминал"
yay -S ttf-roboto ttf-jetbrains-mono
```
| Параметр | Значение | 
| ----- | ----------- | 
| Текст интерфейса | Roboto Regular | 
| Текст документа | Roboto Regular | 
| Моноширенный текст | JetBrains Mono Regular |

### Внешний вид
Я для Gnome юзаю курсор и иконки от WhiteSur, а тему Colloid. Все это есть уже в Aur:

```sh title="Курсор"
yay -S whitesur-cursor-theme-git
```
```sh title="Иконки"
yay -S whitesur-icon-theme
```
```sh title="Темы"
yay -S colloid-dracula-theme-git
```
