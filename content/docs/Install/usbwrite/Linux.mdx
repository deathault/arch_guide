Так, у нас имеется скачаный iso образ с дистрибутивом. Давайте его закатаем на флеху, внешний жесткий диск или че там у вас есть под рукой.

Писать будем через утилиту `dd`. Сначала давай проверим, в какую влажную дырку ты воткнул свою волосатую флешку для записи. Ищем в терминале так: 
```sh title="Теминал"
lsblk
```
Запомните эту утилиту, она достаточно часто вам будет требоваться для работы с внешними устройствами.

Обычно она будет называться шота вроде `/dev/sdX`, где `X` — буква, например, sdb, sdc и т.д. Будь внимателен, чтобы не перепутать флешку с другим устройством.
А то фоточки твоего ребенка, скрупулезно собранные супругой, будут в тот час преданы анафеме, и пизды получишь ты, а не я.
![img](/lsblk.png)

Вот и наша флэха, видим ее имя - `sda1`. Запоминаем. 

Теперь выясняем где лежит наш образ с арчем. Допустим валяется он в загрузках, можем написать sudo dd if= а дальше просто перетащить образ в терминал, он спарсит путь к нему.
```sh title="Теминал"
sudo dd if='/home/ewo/Загрузки/archlinux-2025.01.01-x86_64.iso' of=/dev/sdaX bs=4M status=progress oflag=sync
```
Здесь собственно замените sdaX на свое имя. 

<Callout>
Для дотошных - "а хули происходит?" Ща покажу:
- `if=` - входной iso файл
- `of=` - выходное устройство
- `bs=4M` - размер блока для записи
- `status=progress` - отображаем прогресс записи
- `oflag=sync` - гарантирует шо блоки будут записаны синхронно
</Callout>
