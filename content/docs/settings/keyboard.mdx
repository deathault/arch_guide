## Раскладка
Делаем смену раскладки по `Alt+Shift`:


```sh title="Теминал"
gsettings set org.gnome.desktop.wm.keybindings switch-input-source-backward "['<Shift>Alt_L']"
&& gsettings set org.gnome.desktop.wm.keybindings switch-input-source "['<Alt>Shift_L']"
```
## Горячие клавиши

В настройках клавиатуры идем в самый низ "Просмотр и изменение комбинаций клавиш", снимки экрана, Сделать снимок экрана интерактивно.
