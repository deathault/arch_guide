## После выключения мышка\наушники не отключают подсветку

Узнаем id устройства которое нам хамит:
```sh title="Терминал"
lsusb
```
Насильно разрешаем вырубать его

```sh title="Терминал"
echo '0' | sudo tee /sys/bus/usb/devices/usbX/power/control
```
*Где usbX — это айдишник*
